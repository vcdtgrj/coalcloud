## 用户口碑怎么说

从社区反馈来看，CoalCloud整体评价属于"稳健中偏正面"这个档次。

速度方面，多个用户测试显示广州移动线路升级后延迟明显下降，丢包率基本可以忽略。香港BGP的CMI线路三网回程延迟在30-50ms区间，表现不错。湖北联通9929那条线路联通用户用起来几乎是直连的感觉。

稳定性方面，大多数人的评价是"该有的时候有，偶尔出小问题"。遇到线路故障，CoalCloud的处理方式是延长服务期或者按剩余价值补偿，在国内小型IDC里算是比较负责任的。

也有人吐槽的点：实名认证流程麻烦（不过这是工信部要求，所有国内IDC都一样）；部分套餐限制条件写得不够清楚；客服响应速度有时不够快。

综合来看，CoalCloud适合"知道自己需要什么、对价格敏感、能接受偶尔的小波折"的用户。不是最顶级的服务，但在这个价位段里，性价比是真的不错。
